# AP EAMCET College Predictor

A web-based AP EAMCET College Predictor built using HTML, CSS, and JavaScript.

## Features

* Rank-based college prediction
* Branch-wise filtering
* Admission chance estimation
* Responsive UI
* College and cutoff information

## Technologies Used

* HTML
* CSS
* JavaScript
* VS Code
